## 배포 환경
- **AWS 배포**(현재 비용 문제로 배포 중단)
  - EC2, RDS, S3 배포 환경 구축
  - EC2를 이용해 서버 배포

## 구현 기능
- **Find Mate 게시판 CRUD**
  - 게시글 CRUD
  - 댓글 CRUD
  - 게시글 검색, 정렬
    - 제목, 태그를 이용한 검색
    - 최신순, 조회수순으로 정렬
- **프로젝트**
  - 프로젝트 참가 수락, 거부
- **알림**
  - 주요 메시지 알림
  - 알림 읽음 처리
